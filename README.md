# Metal Quiz – Pythonprojekt

## Kort beskrivning av projektet
Detta projekt är ett quizspel om heavy metal-musik, skrivet i Python.  
Spelet kan köras både i terminalen och som ett grafiskt gränssnitt (GUI) med Tkinter.  
Projektet innehåller även automatiska tester skrivna med pytest för att testa programmets logik.
